# Astrolabe

Astrolabe is an AR app for space debris created for the Nasa Space Apps hackathon

## run for the first time

`npm install `

## run for Qr code

`npm start`

## run for Android emulator

`npm run android`

## run for web

`npm run web`

# contribution
